# 开发环境



## VS Code + Claude Code + GLM Coding Plan

1. [Claude Code：在 VS Code 中使用 Claude Code](https://code.claude.com/docs/zh-CN/vs-code)；（可以不用安装 Claude Code 只安装插件，两者功能有一定[区别](https://code.claude.com/docs/zh-CN/vs-code#vs-code-扩展与-claude-code-cli)）

2. [智谱：Claude Code IDE 插件 配置 GLM Coding Plan](https://docs.bigmodel.cn/cn/coding-plan/tool/claude-for-ide)；

3. [指定模型](https://docs.bigmodel.cn/cn/coding-plan/tool/claude#如何切换使用模型)（按照官方指导，有了前面配置，指定模型的配置是非必要的，但是我这里不指定还是会使用 claude code 的默认模型）

   ```json
   // ~/.claude/settings.json
   {
     "env": {
       "ANTHROPIC_DEFAULT_HAIKU_MODEL": "glm-4.5-air",
       "ANTHROPIC_DEFAULT_SONNET_MODEL": "glm-4.7",
       "ANTHROPIC_DEFAULT_OPUS_MODEL": "glm-4.7"
     }
   }
   ```
